# LinkedIn Job Scraper

A Python tool to scrape job listings from LinkedIn using Selenium, Beautiful Soup, and Pandas.

## Requirements

- Python 3.x
- Chrome browser

## Installation

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

Run the script:
```
python linkedin_job_scraper.py
```

Enter your LinkedIn email, password, and job keywords when prompted.

The results will be saved to `jobs.csv`.

## Features

- Automated login to LinkedIn
- Search jobs by keywords
- Extract job title, company, location
- Save to CSV

**Note:** Scraping LinkedIn may violate their terms of service. Use responsibly.
